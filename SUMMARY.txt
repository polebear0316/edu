[심플 무지개 테트리스 - 프로젝트 작업 요약]

1. 프로젝트 개요
   - 웹 브라우저에서 실행 가능한 심플하고 직관적인 테트리스 게임.
   - PC와 모바일 환경을 모두 지원하는 반응형 웹 어플리케이션.
   - 백엔드 연동을 통한 사용자별 최고 점수 관리 및 리더보드 제공.

2. 주요 구현 기능
   가. 모바일 최적화 (Mobile Optimization)
       - 브라우저 크기에 따라 게임판 크기가 자동으로 조절되는 동적 리사이징.
       - 모바일 터치 환경을 고려한 D-pad(십자키) 스타일의 조작 버튼 레이아웃.
       - 터치 지연 시간을 최소화하기 위한 Pointer Event 기반의 조작계.

   나. 사용자 인터페이스 (UI)
       - 게임 시작 전 사용자 이름을 입력받는 기능 추가.
       - 조작법 안내 및 실시간 TOP 10 랭킹(리더보드) 오버레이 제공.
       - 게임 오버 시 서버에 점수를 자동 기록하고 랭킹을 갱신.

   다. 백엔드 및 데이터베이스 (Backend & DB)
       - Node.js와 Express.js를 사용한 RESTful API 서버 구축.
       - JSON 기반의 경량 데이터베이스(database.json)를 통한 영구 저장.
       - 주요 API: 사용자 등록(/register), 점수 기록(/score), 랭킹 조회(/leaderboard).

   라. 게임 로직 (Core Logic)
       - 7종의 테트로미노 및 무지개 색상 매핑.
       - 정교한 충돌 감지 및 라인 삭제 로직.
       - 하드 드랍(Hard Drop) 기능 지원.

3. 파일 구조
   - index.html, style.css, script.js: 프론트엔드 핵심 파일
   - server/index.js: 백엔드 서버 메인
   - server/db.js: JSON 데이터베이스 관리 로직
   - server/database.json: 사용자 및 점수 데이터 저장소
   - SUMMARY.txt: 프로젝트 작업 요약 문서
   - TEST_PLAN.md, test-logic.js, test-runner.html: 테스트 관련 파일

4. 향후 확장 계획 (제안)
   - 레벨 시스템 (점수에 따른 속도 증가)
   - 음향 효과 및 배경음악 추가
   - 블록 홀드(Hold) 기능 추가
